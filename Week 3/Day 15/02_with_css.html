Что такое CSS?

CSS позволяет управлять внешним видом HTML элементов. Можно изменять цвета, размеры, отступы и многое другое.

Основные свойства

  • color - цвет текста
  • background-color - цвет фона
  • padding - внутренний отступ
  • margin - внешний отступ
  • border - рамка

Таблица с данными

Свойство Описание
font-size Размер шрифта
text-align Выравнивание текста
border-radius Скругление углов
